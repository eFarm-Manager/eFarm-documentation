# 1. GET landparcels list
* GET
* path: ```/api/landparcel/all```
* Required role: ```"ROLE_FARM_OWNER"``` or ```ROLE_FARM_MANAGER```
* Example body: NONE
* Request param:
  - Optional String: ```searchString```
  - Optional Double: ```minArea```
  - Optional Double: ```maxArea```
