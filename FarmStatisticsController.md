# 1. GET crop area statistics
* GET
* path: ```/api/statistics/crop-area```
* Required role: ```"ROLE_FARM_OWNER"``` or ```"ROLE_FARM_MANAGER"```
* Example body: NONE
* Request param:
  - Optional String: ```seasonName```
