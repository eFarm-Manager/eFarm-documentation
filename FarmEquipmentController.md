# 1. GET all farm equipment
* GET
* path: ```/api/equipment/all```
* Required role: ```"ROLE_FARM_OWNER"``` or ```ROLE_FARM_MANAGER```
* Example body: NONE
* Request param:
  - Optional string: ```searchQuery```
  - Optional boolean: ```onlyAvailable```
